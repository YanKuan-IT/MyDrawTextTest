
## DrawText 文本绘制效果测试
## 变色效果
1）先对文字区域，做裁剪，裁剪出来两个矩形，分别在两个矩形区域绘制文本内容及颜色；
2）通过动画，持续改变矩形的区域，打到变色的效果
